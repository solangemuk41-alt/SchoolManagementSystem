 🎓 School Management System

A simple Java project using the **Spring Framework** to demonstrate DI, IoC, and Loose Coupling.

 📁 Classes
- `Person` — base class with name, age, email
- `Student` — extends Person, adds studentId and course
- `Teacher` — name, subject, experience
- `Course` — links to a Teacher
- `Enrollment` — enrolls a student into a course

 🌱 Spring Annotations Used
| Annotation | Purpose |
|---|---|
| `@Configuration` | Marks the config class |
| `@Bean` | Creates beans manually |
| `@Component` | Auto-detects beans |
| `@Autowired` | Injects dependencies |

 ▶️ How to Run
1. Clone the repo
2. Open in IntelliJ
3. Run `Main.java`
